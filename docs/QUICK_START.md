# Quick Start Guide - Zebra Printing

## 🚀 First Time Setup (One-Time Only)

### 1️⃣ Install Browser Print

- Download from Zebra website
- Install like any other program
- It runs in the background (you won't see a window)

### 2️⃣ Accept Security Certificate

- Open Chrome/Firefox
- Go to: **https://localhost:9101/ssl_support**
- Click through the security warning (this is safe!)
- See "SSL Support Enabled" ✅

## 📋 Daily Use - Printing Labels

### 1️⃣ Open the Web App

```
🌐 → Your browser → Bookmark or type the URL
```

### 2️⃣ Check the Status

Look for the status box:

- 🟢 "Ready to print" = Good to go!
- 🔴 "Browser Print not detected" = See setup above

### 3️⃣ Pick Your Printer

```
📍 Dropdown Menu → Select your location's printer
   ↓
📄 Printer details appear
```

### 4️⃣ Load Your Label

Choose one:

- 📁 **Upload**: Click "Choose File" → Select .zpl file
- 📋 **Paste**: Copy/paste ZPL code into the box
- 🧪 **Sample**: Click "Load Sample" for testing

### 5️⃣ Print!

```
🖨️ Click "Print Label" → Watch status → ✅ Success!
```

## ⚡ Quick Troubleshooting

| Problem                       | Quick Fix                                                                       |
| ----------------------------- | ------------------------------------------------------------------------------- |
| 🔴 Browser Print not detected | 1. Restart Browser Print<br>2. Refresh web page<br>3. Re-accept SSL certificate |
| 🔴 Can't connect to printer   | 1. Check printer is ON<br>2. Verify on same network<br>3. Try "Test Connection" |
| 🔴 Nothing prints             | 1. Check for paper<br>2. Check printer not paused<br>3. Try sample label        |

## 💡 Pro Tips

1. **Bookmark the app** for quick access
2. **Save working ZPL files** for reuse
3. **Test connection first** when using a new printer
4. **Use Chrome** for best results (not Edge!)

## 🆘 Need Help?

1. Try the quick fixes above
2. Restart Browser Print:
   - Windows: System tray → Right-click → Exit → Restart
   - Mac: Menu bar → Quit → Reopen from Applications
3. Call IT with:
   - Error message screenshot
   - Printer name
   - What you were trying to do
