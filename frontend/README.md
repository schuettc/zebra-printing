# Frontend Application (`frontend/`)

This directory contains the Vite React TypeScript application for the Zebra Printing project.

## Key Files

- `src/main.tsx`: The entry point for the React application.
- `src/App.tsx`: The root React component.
- `index.html`: The main HTML file that loads the React app.
- `vite.config.ts`: Vite configuration file.
- `package.json`: Manages dependencies and scripts for this frontend project.
- `tsconfig.json`: TypeScript configuration.

## Available Scripts

(Run from the `frontend/` directory or using `pnpm --filter frontend <script>` from the root)

- `pnpm dev`: Starts the Vite development server (typically on `http://localhost:5173`).
- `pnpm build`: Builds the application for production (output in `frontend/dist/`).
- `pnpm preview`: Serves the production build locally for preview.
- `pnpm lint`: Lints the codebase using ESLint.
- `pnpm format`: Formats the codebase using Prettier.

Refer to the [root README.md](../README.md) for more details on project setup and overall architecture.
