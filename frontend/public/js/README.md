# Browser Print SDK Files

This directory should contain the Zebra Browser Print SDK files:

1. **BrowserPrint-3.1.250.min.js** - Core Browser Print SDK
2. **BrowserPrint-Zebra-1.1.250.min.js** - Zebra-specific extensions

## Download Instructions

1. Visit the Zebra Developer Portal: https://developer.zebra.com/
2. Navigate to the Browser Print section
3. Download the JavaScript SDK files (may require registration)
4. Place the downloaded files in this directory

## File Verification

The files should have the following approximate sizes:

- BrowserPrint-3.1.250.min.js: ~50KB
- BrowserPrint-Zebra-1.1.250.min.js: ~15KB

Note: Version numbers may vary. Update the references in index.html if using different versions.
